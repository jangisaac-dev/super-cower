# Super Cower Docs

Use this directory for repository-local planning artifacts created by
`super-cower`.

- Approved designs: `docs/super-cower/specs/YYYY-MM-DD--design.md`
- Executable implementation plans: `docs/super-cower/plans/YYYY-MM-DD--plan.md`

Keep these documents concise, decision-complete, and aligned with the task-packet
contracts described under `references/`.
